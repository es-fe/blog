# 文档编写指南

  1. 拉取代码： git clone https://github.com/es-fe/blog.git

  2. 分支切换到main

  3. 安装依赖： yarn

  4. 在docs文件夹目录下新建md文件。

  5. 在.vuepress/config.js配置路由。

  6. 开发模式查看： yarn docs:dev

  7. 发布： yarn deploy
